Konfigurasi Rule Firebase

Salin aturan keamanan (Security Rules) di bawah ini dan tempelkan pada tab Rules di Firestore Database konsol Firebase Anda. Ini wajib dilakukan agar sinkronisasi Cloud Hub berjalan tanpa terblokir sistem.

firestore.rules 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Memastikan koneksi database EUNOIA aman
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
Langkah Pemasangan: 1. Buka Konsol Firebase lalu pilih project Anda.
2. Masuk ke menu Firestore Database di bilah navigasi kiri.
3. Klik tab Rules (Aturan) yang berada di sebelah tab Data.
4. Hapus semua kode lama, lalu Paste (Tempel) kode yang telah disalin di atas.
5. Klik tombol Publish (Publikasikan).
